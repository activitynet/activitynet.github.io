Task B– Kinetics-AVA Crossover challenge

Challenge 2020 → Task B - Kinetics-AVA Crossover challenge

The AVA-Kinetics task is an umbrella for a crossover of the previous AVA and Kinetics tasks, where Kinetics has now been annotated with AVA labels (but AVA has not been annotated with Kinetics labels). There has always been some interactions between the two datasets, e.g. many of the AVA methods are pre-trained on Kinetics. The new annotations should allow for improved performance on both tasks and also increase the diversity of the AVA evaluation set (which now also includes Kinetics clips).

Dataset

The AVA-Kinetics dataset is a combination of AVA and Kinetics-700 datasets where the full validation and test splits of Kinetics-700 and a significant part of its training set now have AVA annotations (bounding boxes and atomic actions).

Evaluation Metric

The evaluation metrics will be the same as for the AVA and Kinetics tasks from previous years.

Baselines

Baselines will be the same as in previous years for AVA and Kinetics.

Submission Format

The submission format will be the same as for the AVA and Kinetics tasks from previous years.
